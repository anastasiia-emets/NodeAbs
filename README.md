# How to use?
- Clone repository: `git clone https://github.com/anastasiia-emets/NodeAds.git`
- Install gulp [npm](https://www.npmjs.com/): `npm install`
- Run the project with a gulp [gulp](https://gulpjs.com/): `npx gulp`
